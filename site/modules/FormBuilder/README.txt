HOW TO INSTALL
==============

1. Form Builder requires ProcessWire 2.2.9 or newer. Make sure you update your ProcessWire
version before installing Form Builder.

2. Unzip and place the files in /site/modules/FormBuilder/. 

3. In the ProcessWire admin, click to Modules > Check for new modules. 

4. Click install for Form Builder. 

5. If instructed to, copy /site/modules/FormBuilder/form-builder.php to
/site/templates/form-builder.php

6. Enter your license key on the Form Builder module configuration screen and submit.

7. Review the configuration options and adjust as necessary, then submit.

8. Go to the Setup menu in your ProcessWire admin and click on Forms to begin using
Form Builder! 


UPGRADES
========

Upgrades are made available through the ProcessWire Form Builder support forum at
http://processwire.com/talk/. To install an upgrade do the following:

1. Replace the entire contents of your /site/modules/FormBuilder/ directory with 
the contents of the new Form Builder zip file. If you have created any custom themes
make sure that you back them up first. 

2. Replace your /site/templates/form-builder.php with the new one in 
/site/modules/FormBuilder/form-builder.php

3. Your Form Builder should now be upgraded. If you had any custom themes, you may
place them back into your /site/modules/FormBuilder/themes/ directory.
